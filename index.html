Go 언어 기반의 대규모 분산 시스템 및 블록체인 연동 서비스 개발 전문가, 김서규입니다. Fault-Tolerant 시스템 설계와 최적화 경험을 바탕으로 안정적이고 확장 가능한 백엔드 솔루션을 제공하며, 특히 블록체인 인프라 연동 및 미들웨어 개발을 통해 실질적인 비즈니스 가치 창출에 기여합니다.

  • Go 기반 대규모 분산 시스템(블록체인 Offchain DB, Fault-Tolerant 시스템, GIS 기반 서비스 등) 설계, 개발 및 최적화 경험
  • 견고하고 테스트 용이성이 높은 백엔드 아키텍처 설계 및 MSA(Microservices Architecture) 환경 구축 역량
  • 제품 중심의 심층적인 문제 해결 능력 및 다양한 직군과의 원활한 커뮤니케이션을 통한 성공적인 협업 경험

기술 스택

프로그래밍 언어

Go: 대규모 분산 MSA 개발 및 운영, 고성능 블록체인 연동 API 서버 구축 주력. 클린 아키텍처 적용 경험.

TypeScript: 거래소 입출금 백엔드, 메타버스 토지 거래 백엔드 개발 경험. 사내 어드민 페이지와 블록체인 연동 프론트엔드 개발 경험.

Python: 테스트 자동화(PyTest), 시스템 스크립트, 사내 미들웨어 및 CI/CD 파이프라인 스크립트 개발 활용.

Rust: Rust 기반 블록체인 인덱서와 블록체인 노드 연동 모듈 개발 경험.

데이터베이스 & 스토리지

  • PostgreSQL (PostGIS): 복잡한 SQL 쿼리 최적화, LBS 데이터 모델링 및 공간 연산, 대용량 DB 튜닝.
  • Redis: 캐싱 전략, Redis Streams 메시지 큐, Pub/Sub, 분산 락.
  • MongoDB: Document 스키마 설계, BSON 최적화, Aggregation Pipeline, Offchain 데이터 저장/인덱싱.
  • AWS S3: 대용량 파일 스토리지, 정적 호스팅, 백업/아카이빙, Presigned URL 보안 접근.

DevOps, 클라우드 & 모니터링

  • Docker & Kubernetes: Containerization, Helm Chart, Istio Service Mesh, Kustomize, CI/CD 자동 배포.
  • GitLab CI: CI/CD 파이프라인 구축/최적화, 자동화된 테스트/빌드/배포.
  • Datadog / Prometheus & Grafana: 통합 모니터링, Log 파싱(Grok), 커스텀 대시보드, SLA/SLO 알림.
