# 롤업이란?

- 롤업은 L2에서 트랜잭션을 실행한 뒤, 실행한 트랜잭션 데이터들과 변경된 상태의 요약본을 L1에 저장하는 솔루션
- 일반적으로 L1에 배패된 롤업 컨트랙트에 스테이트 루트(상태요약본)가 저장되어 있음
- 롤업의 경우 L1의 상태 변경시 TX실행 결과 뿐만 아니라 데이터 또한 보관하는 방식을 통해서 누구든 데이터 유효성을 검증할 수 있게 했음
